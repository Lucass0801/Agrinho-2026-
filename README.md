# 🌱 Agrinho - Sustentabilidade no Campo

Esse projeto foi feito para o Agrinho com o tema **Sustentabilidade no Campo**.

A ideia do site é mostrar como a agricultura pode ajudar o meio ambiente usando práticas sustentáveis e tecnologia no campo.

---

## 💻 O que tem no site?

- Página inicial moderna
- Banner com imagem do campo
- Cards informativos
- Seção sobre sustentabilidade
- Práticas sustentáveis
- Site responsivo para celular

---

## 🚀 Tecnologias usadas

- HTML
- CSS
- VS Code
